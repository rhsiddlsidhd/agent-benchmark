### 수정 필요
- `src/lib/tmdb/client.ts:123` — `return (await response.json()) as T;` 의 `as T` 단언이 전역 CLAUDE.md typing 규칙("`as const` 제외 `as` 단언 금지")을 위반. any 오염은 없으나 명시적 금지 대상.
  - 수정 방향(단순, 무의존성): `response.json()`이 `any`를 반환하므로 단언 없이 타입 변수 대입으로 대체 가능 —
    ```ts
    const data: T = await response.json();
    return data;
    ```
    `any`는 `T`에 할당 가능하므로 `as` 없이 동일 결과. (런타임 검증까지 원하면 zod 등 추가는 사용자 승인 필요 → 스코프 밖, 본 FIX 범위 아님.)
