- **[환경] node_modules 미설치 → build/lint/typecheck 미실행.**
  `npm ci` 안 된 상태라 T1 완료기준의 `next build`/`lint` 통과를 실제 실행으로 확인하지 못함(승인 없이 패키지 설치 안 함). 대상 파일 4개는 정적상 타입 에러 요소 없음(단순 컴포넌트/CSS)이라 코드 결함 아님. 의존성 설치 후 `npm run build && npm run lint` 재확인 권장 — 이 항목은 **미검증**으로 남긴다.

- **[T4 이월] `text-base` 유틸 충돌 위험 — page.tsx:18.**
  `--color-base` 정의로 `bg-base`/`border-base`는 무충돌이나, `text-base`는 Tailwind v4 **기본 폰트사이즈 유틸(--text-base:1rem)**과 이름이 겹친다. page.tsx:18 CTA의 `text-base`는 어두운 base 색을 의도한 것으로 보이나 폰트사이즈로 해석되면 브랜드 시안 배경 위 텍스트가 밝은색으로 남아 의도한 대비가 안 나올 수 있음. page.tsx는 T1 범위 밖(T4에서 교체) 플레이스홀더라 T1 판정에는 영향 없음. **수정 방향(T4):** 온-브랜드 텍스트는 명시적 색 토큰(예: `text-content-primary` 또는 별도 on-brand 토큰) 사용, `text-base`로 색을 지정하지 말 것.
