## 필드 가이드

| 섹션 | 내용 |
|------|------|
| 배경 | 이 프로젝트/기능이 왜 필요한가, 현재 어떤 문제가 있는가 |
| 목표 | 무엇을 달성하려 하는가 (측정 가능하게), Non-Goals: 명시적으로 제외하는 것 |
| 대상 사용자 | 누가 사용하는가, 사용자 페르소나/시나리오 |
| 요구사항 | 기능(FR) / 비기능(NFR) 요구사항, 우선순위 |
| 사용자 스토리 | As a [사용자], I want [기능], so that [목적] |
| 성공 지표 | KPI / 측정 방법 |
| 제약사항 | 기술적, 일정, 리소스 제약 |
| 오픈 이슈 | 미결정 사항 |
